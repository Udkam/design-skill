# Brief: Mini Program

Use `$craft-frontend-design` to design a WeChat mini-program-style flow for booking a community study room. It should be fast, Chinese-facing, mobile-first, and suitable for short sessions.

## Expected Routing

- `interface-taxonomy.md`: WeChat mini-program-style flow.
- `platform-adapters.md`: mini-program constraints.
- `content-and-microcopy.md`: concise Chinese copy.
- `quality-gates.md`: touch target and state coverage.

## Expected Output

- Starts with date/room availability.
- Uses shallow navigation and thumb-friendly controls.
- Covers loading, empty, error, disabled, unavailable, selected, and confirmation states.
- Avoids desktop landing-page layout.
