# Human Rubric

Score 0-2 for each category.

- Personal identity: keeps a specific author surface instead of a generic landing-page template.
- Project navigation: Projects, Game, Repo, and Console entries are clear and reachable.
- Pet/toy quality: interaction feels optional, bounded, and helpful rather than dominant.
- Motion restraint: motion has a purpose and JS loops stop in reduced-motion mode.
- Responsive robustness: no horizontal overflow; dock and pet controls avoid bottom overlap.
- Accessibility: focus, labels, alt text, disabled states, and text contrast are covered.
- Shared component safety: changes to islands, docks, theme controllers, or pet controls include a blast-radius note.
