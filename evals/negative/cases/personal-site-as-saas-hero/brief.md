# Brief: Negative - Personal Site As SaaS Hero

This synthetic fixture represents a personal site redesign that lost its identity and became a generic product-marketing page.

## Expected Problems

- Personal site is converted into a purple/blue SaaS hero.
- Pet/toy identity is deleted or reduced to a decorative floating control.
- Projects, Game, Repo, and Console entries are hidden behind generic feature cards.
- Reduced motion only changes CSS; a JS `requestAnimationFrame` loop continues.
- Mobile floating controls overlap the route dock.
- Shared component blast radius is not addressed in the design narrative.
