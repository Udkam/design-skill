# Inspiration Sampling Playbook

Use this before referencing award sites, prompt galleries, component libraries, animation galleries, GSAP resources, or competitor/product sites.

## Rule

References are mechanism sources, not copy sources.

For each reference, record:

```text
Reference:
Extracted mechanism:
Forbidden to copy:
Transferable principle:
Implementation effect:
```

## Mechanisms To Extract

- information rhythm;
- density changes;
- proof placement;
- visual thesis;
- type/spacing relationship;
- interaction cause;
- motion pacing;
- asset role;
- mobile adaptation;
- performance restraint;
- accessibility compromise to avoid.

## Never Copy

- full layouts;
- section order from one site;
- text, slogans, labels, or claims;
- imagery, screenshots, icons, mascots;
- distinctive animation chains;
- proprietary component combinations;
- recognizable compositions.

## Source-Specific Use

- Awwwards / CSS Design Awards / FWA / Webby:
  extract craft mechanisms, composition discipline, interaction purpose, and proof rhythm.
- Landing.Love:
  extract hero clarity, conversion flow, content cadence, and responsive section treatment.
- React Bits:
  extract implementation ideas for animated text/background/microinteraction, then simplify and add reduced motion.
- 21st.dev:
  extract component structure and interaction variants, not visual identity.
- GSAP AI Skills / GSAP docs:
  extract correct timeline, ScrollTrigger, plugin, cleanup, and framework integration patterns.
- Google Fonts:
  extract font pairing, variable font, language support, and readability considerations.
- WeUI / mini-program examples:
  extract platform conventions and task-flow structure.

## Sampling Output Example

```text
Reference: Awwwards editorial product page group.
Extracted mechanism: Alternating dense proof blocks with quiet image-led pauses.
Forbidden to copy: Exact section ordering, image treatment, and headline rhythm.
Transferable principle: Give users visual rest after proof-heavy sections.
Implementation effect: Insert a compact evidence table followed by one calm artifact preview.
```

Do not claim the result is award-winning unless it actually won an award.
